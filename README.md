üèüÔ∏è PLAYGROUND - Sistem Informasi Booking Lapangan

Sistem informasi berbasis web yang dirancang untuk memudahkan manajemen penyewaan lapangan olahraga (Futsal/Badminton/Voli) secara digital. Aplikasi ini mendukung multi-mitra (cabang) dengan dashboard admin yang informatif.

üöÄ Fitur Utama

**üë§ Sisi Pengunjung (User)**
- **Pemesanan Online:** Booking lapangan sesuai jadwal yang tersedia.
- **Cek Status Lapangan:** Mengetahui lapangan mana yang sedang tersedia atau dalam masa renovasi.
- **Upload Bukti Bayar:** Fitur konfirmasi pembayaran dengan unggah foto bukti transfer.

**üõ†Ô∏è Sisi Admin**
- **Dashboard Statistik:** Ringkasan total booking, pesanan pending, dan pesanan lunas.
- **Manajemen Booking:** Verifikasi pembayaran (set Lunas) dan hapus pesanan.
- **Manajemen Fasilitas:** Mengelola data lapangan dan status operasional (Tersedia/Renovasi).
- **Laporan Cetak:** Ekspor data booking ke format **PDF** dan **Excel**.

## üíª Teknologi yang Digunakan
- **Bahasa:** PHP 8.x
- **Database:** MySQL
- **Frontend:** Bootstrap 5, Font Awesome 6
- **Interaksi:** SweetAlert2 (Notifikasi Modern)
- **Fonts:** Poppins (Google Fonts)
